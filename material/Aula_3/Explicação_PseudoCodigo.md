> Insertion-Sort(A, n)

```
forj = 2 to n
  key = A[j]
  // Insert A[j] into the sorted sequence
  i = j - i
  while i > 0 and A[i} > key
    A[i + 1] = A[i]
    i = i - 1
  A[i + j] = key
```
